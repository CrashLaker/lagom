# lagom
lagom: a simplistic presentation generator

## installation

```
npm install lagom -g --production
```

## usage

### create a presentation
```
lagom create
```

### work on a presentation / do a presentation
```
cd <lagom project folder>
lagom server
```

### deploy presentation (work in progress)
```
lagom deploy

```
